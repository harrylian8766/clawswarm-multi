# ClawSwarm Document Read

Use the `clawswarm_read_document` tool to read a ClawSwarm document by URI.

## Tool

Call `clawswarm_read_document` with:

```json
{
  "uri": "clawswarm://projects/<project-id>/documents/<document-id>"
}
```

## URI Example

```text
clawswarm://projects/d7e341a3-30de-40be-9545-f39cc1bddca8/documents/c9dc2a9c-19ea-41d2-a35e-b94871301190
```
